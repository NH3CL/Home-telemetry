# Home-telemetry
Home telemetry system by NH3CL

My Public ThingSpeak channel: https://thingspeak.com/channels/929511


# Components

 - Central station
   - Raspberry Pi 3B+
 - Nodes
   - NodeMCU ESP8266 ESP-12E (5V powered)
 - PSU
   - Power bank
